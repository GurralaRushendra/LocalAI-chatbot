# Local RAG Chatbot

A local Retrieval-Augmented Generation (RAG) chatbot built using LangChain, Chroma, and Ollama.

## Overview
- Converts restaurant reviews into embeddings
- Stores them in a vector database
- Retrieves relevant context using semantic search
- Generates grounded responses

## Tech Stack
- Python
- LangChain
- Chroma
- Ollama
- Pandas
